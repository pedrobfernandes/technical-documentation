Introduction

HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets) are the foundational technologies for building web pages. HTML structures the content, while CSS styles it, allowing developers to create visually appealing and well-organized websites.

What is HTML?

HTML is the standard markup language used to create web pages. It describes the structure of a webpage using elements and tags, allowing browsers to interpret and display content effectively.

Key Features:

  • Structure: Organizes content into elements like headings, paragraphs, links, and lists
  • Semantics: Provides meaning to web content, enhancing accessibility and SEO.

Basic HTML Structure

An HTML document typically follows this structure:


<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Explanation of Key Tags:

  • <!DOCTYPE html>: Declares the document type.
  • <html>: Root element of the HTML document.
  • <head>: Contains meta-information about the document.
  • <body>: Contains the content of the document.

Common HTML ELements

Here are some commonly used HTML elements:

  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Links: <a href="URL">Link text</a>
  • Images: <img src="image.jpg" alt="description">
  • Lists: <ul>, <ol>, and <li>
  • Forms: <form>, <input>, <button>

What is CSS?

CSS is a stylesheet language used to describe the presentation of HTML documents. It enables developers to control layout, colors, fonts, and overall visual aesthetics of web pages.

Key Features:

  • Separation of Concerns: Keeps content (HTML) and presentation (CSS) separate for easier maintenance.
  • Responsive Design: Allows for media queries and flexible layouts that adapt to different screen sizes.

CSS Syntax and Selectors

CSS rules consist of selectors and declarations that use properties and values. Example:

h1 { color: blue; font-size: 24px; }

Where:

  • h1: This is the selector
  • color: This is the property
  • blue: This is the value

Types of Selectors:

Here is a list of the types of selectors you can use:

  • Type Selector: Selects all elements of a given type (e.g., p)
  • Class Selector: Selects elements with a specific class (e.g., .className)
  • ID Selector: Selects a unique element (e.g., #idName)
  • Attribute Selector: Selects elements based on attributes (e.g., [type="text"])

Common CSS Properties

Some common css properties you will use often:

  • Color: color, background-color
  • Font: font-family, font-size, font-weight
  • Box Model: margin, padding, border
  • Layout: display, position, flex, grid

Best Practices

When writing HTML and CSS keep this list of best practices in mind:

  • Use Semantic HTML: Use appropriate tags to enhance readability and SEO.
  • Keep CSS Organized: Use comments, grouping, and consistent naming conventions.
  • Minimize Inline Styles: Prefer external stylesheets for better maintainability.
  • Test Responsiveness: Use media queries to ensure designs look good on all devices.
